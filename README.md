**☘게임 소개** 

환경 문제의 심각성을 게임을 통해 알려주고자, 짧지만 임팩트를 주고 싶어서 제작하게 된 미니 게임입니다. 평범한 대학생인 김눈송은 눈을 떠보니 이상한 건물 안에 갇히게 되어 소원을 들어주면 원래 있던 곳으로 돌려보내준다는 소리를 듣게 됩니다. 그렇게 건물에서 탈출하기 위해 소원을 이뤄주려 모험을 떠나는 미니게임 형식의 탈출 게임입니다.
